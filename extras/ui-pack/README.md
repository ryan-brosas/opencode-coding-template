# ui-pack

Optional visual design, Figma, image generation, browser automation, mockup extraction, and style-direction workflows.

Frontend implementation, accessibility auditing, UX quality gates, React/front-end patterns, and the `vision` review agent are part of the default coding core. Keep this pack for workflows that are closer to design/asset production than coding.

Copy reviewed files from this pack into the matching `.opencode/` subdirectories to enable them.
