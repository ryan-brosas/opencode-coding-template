# ui-pack

Optional visual review, design, Figma, image generation, browser automation, mockup extraction, and style-direction workflows.

Frontend implementation quality remains in the default coding core through `frontend-implementation-quality`, `accessibility-audit`, `ux-quality-gates`, and `react-best-practices`. Use this pack for workflows that are closer to visual critique, design/asset production, mockup/Figma extraction, or browser automation than core coding checks.

Copy reviewed files from this pack into the matching `.opencode/` subdirectories to enable them.
